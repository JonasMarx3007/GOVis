GOVIS - LOCAL INSTALL AND RUN

Requirements
- Python 3.11+
- Node.js LTS

Install
- Open a terminal in the project folder
- Run:

  build.cmd

Run
- In cmd:

  run

- In PowerShell:

  .\run.ps1

Open
- http://127.0.0.1:8000/

Direct Python run
- Optional:

  python -m backend --host 127.0.0.1 --port 8000
